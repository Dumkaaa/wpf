# WPF
WPF is a WordPress Framework to make your creating of theme easy.

## Requires
* PHP v7.*
* Wordpress
* Advanced Custom Fields (ACF) Wordpress Plugin

See [Wiki](https://github.com/nikonorovsv/wpf/wiki) to find out how this works.
